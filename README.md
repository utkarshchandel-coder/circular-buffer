# Circular Buffer in C

## Overview

This project implements a circular buffer (ring buffer) in C, commonly used in embedded systems for efficient data handling.

## Features

* Fixed-size buffer (256 bytes)
* O(1) insertion and removal
* Lightweight and memory-efficient
* Suitable for embedded firmware

Data: 20

## File Structure

* circular_buffer.h → Structure and function declarations
* circular_buffer.c → Implementation
* main.c → Test program
